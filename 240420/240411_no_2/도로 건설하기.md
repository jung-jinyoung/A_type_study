# 문제

가로 W 세로 H 개의 격자점    5 ≤ W, H ≤ 50 <br>
직사각형 모양의 마을<br>
도로로부터 가장 멀리 떨어진 집과의 거리를 최소화<br>

도로는 하나만 건설<br>
직선으로만 건설, 마을 끝에서 끝까지 가로지른다.<br>
가로, 세로, 대각선 방향 가능    
집이 있는 위치에는 도로 건설 x     
대각선 방향의 도로는 최소 2 개 이상의 격자점을 지나야 한다.   
<br>
도로로부터 가장 멀리 떨어진 집과의 거리가 최소가 되는 경우를 찾고, 이때 거리를 출력    
도로를 건설할 수 없을 경우 -1을 출력    


## 입력
T : 테케 수         
W 가로 , H 세로          
가로x세로 2차원 Map           

## 출력
도로로부터 가장 멀리 떨어진 집과의 거리가 최소가 되는 경우를 찾고, 이때 거리를 출력        
도로를 건설할 수 없을 경우 -1을 출력
